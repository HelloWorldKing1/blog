## 使用
常用MarkdownPad2快捷键：<br>

|  快捷键   | 说明  |
|  :------:  | :------:  |
| Ctrl + B  | 粗体 |
| Ctrl + I  | 斜体 |
| Ctrl + Q  | 引用 |
| Ctrl + K  | 代码块 |
| Ctrl + 1  | 标题1 |
| Ctrl + 2  | 标题2 |
| Ctrl + 3  | 标题3 |
| Ctrl + U  | 无序列表 |
| Ctrl + Shift + O  | 有序列表 |
| Ctrl + G  | 图片 |
| Ctrl + L  | 超链接 |
| Ctrl + T  | 时间戳 |
| Ctrl + R  | 水平标尺 |
